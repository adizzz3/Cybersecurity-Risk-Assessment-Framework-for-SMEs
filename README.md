# Cybersecurity Risk Assessment Framework for Small Businesses

## Overview

This project presents a cybersecurity risk assessment framework for Small and Medium Enterprises (SMEs). The assessment identifies organizational assets, evaluates cyber threats and vulnerabilities, calculates risks, and recommends mitigation strategies based on industry best practices.

---

## Objectives

- Identify critical business assets
- Analyze cyber threats
- Perform vulnerability assessment
- Evaluate cybersecurity risks
- Recommend mitigation strategies
- Improve organizational security posture

---

## Methodology

The project follows these steps:

1. Asset Identification
2. Threat Identification
3. Vulnerability Assessment
4. Risk Analysis
5. Risk Matrix
6. Risk Register
7. Security Recommendations

---

## Framework Used

- NIST Cybersecurity Framework (CSF)

Core Functions:

- Protect
